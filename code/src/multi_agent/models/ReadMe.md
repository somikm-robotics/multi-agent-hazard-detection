# Third-Party Models

The following Gazebo models are required but not included in this repository.
Download and place in the `models/obstacles/` directory before running.

| Model | Author | Source |
|-------|--------|--------|
| `medium_rock_fall` | Open Robotics | [Gazebo Model Database](https://github.com/osrf/gazebo_models) |
| `small_rock_fall` | Open Robotics | [Gazebo Model Database](https://github.com/osrf/gazebo_models) |
| `mossy_bark` | Milekium | [Gazebo Fuel](https://app.gazebosim.org/fuel) |
| `wood_case` | Milekium | [Gazebo Fuel](https://app.gazebosim.org/fuel) |
| `office_building` | Open Robotics | [Gazebo Model Database](https://github.com/osrf/gazebo_models) |
